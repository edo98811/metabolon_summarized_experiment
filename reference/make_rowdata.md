# Clean and format the row data from the CDT file

Clean and format the row data from the CDT file

## Usage

``` r
make_rowdata(rowdata, rowdata_key)
```

## Arguments

- rowdata:

  A data frame containing the row data from the CDT file.

- rowdata_key:

  A character string specifying the column name to use as row names.

## Value

A cleaned and formatted row data data frame.
